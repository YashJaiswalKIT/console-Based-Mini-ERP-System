# console-Based-Mini-ERP-System
🎓 Mini College ERP System (Console Based)

A Console Based Mini College ERP System built using Java and File Handling.

This project simulates the core structure of a real ERP system used in colleges, focusing on role-based access control and data management without using any external database.

🚀 Features
👑 Super Admin

Add new colleges

Manage college credentials

🏫 College SubAdmin

Add Faculty

Add Students section-wise (A, B, etc.)

Manage academic structure

👨‍🏫 Faculty

Mark attendance section-wise

🎓 Student

View attendance report (Present / Absent count)

🗂️ File-Based Database Structure

Instead of MySQL or MongoDB, this project uses text files as a database:

collegeData.txt
facultyData.txt
studentData.txt
attendance.txt

This helps in understanding:

Java File Handling

BufferedReader / FileWriter

Data filtering logic

Role-based system design

🛠️ Technologies Used

Java

OOP Concepts

File Handling

Console-Based UI
